01Проблема та аудиторія

Проблема

Фрілансери трекають час по кількох клієнтах і потребують швидко затрекати (1–2 тапи), бачити час+гроші, звіти за період та інвойси зі статусами оплати.

Користувач

Фрілансер з кількома клієнтами, працює переважно з телефону, цінує швидкість і простоту понад усе.

Наш кут

Mobile-first, мінімалізм, максимально простий ручний трекінг + інвойси. Local-first зараз → синхронізація потім.

02Конкуренти — 3 групи

До 5 у групі. Для кожного: тип · чому в групі · що вивчити для нас.

ХАРД той самий продукт, аудиторія, ринок

  • Harvest — time tracking + invoicing. Класика «час→ставка→інвойс». Вивчити: потік запис→інвойс, модель ставок, статуси.
  • Clockify — free time tracker. Велика фрілансерська база, мобільний. Вивчити: простота старту таймера, billable vs non.
  • Paymo — freelancer PM + time + invoice. Близько до Клієнт→Проєкт. Вивчити: ієрархія, час→інвойс, звіти.
  • TimeCamp — time tracking + billing. Ставки, білабельність. Вивчити: логіка rate та оверрайдів.
  • Timely — mobile-first / auto tracking. Вивчити: мобільні патерни швидкого трекінгу.

СОФТ інший продукт, той самий JTBD (трекати роботу)

  • Trello — kanban. Колонки-статуси ≈ наші не сплачено/відправлено/сплачено. Вивчити: kanban-статуси, швидке додавання.
  • Asana — task/project mgmt. Вивчити: структура проєктів без перевантаження.
  • Jira — issue tracking. Антиприклад складності. Вивчити: де складність вбиває швидкість; фільтри.
  • Notion — docs + DB. Вивчити: view-патерни (таблиця/борд/календар) для звітів.
  • Todoist — task manager. Еталон швидкого вводу. Вивчити: quick-add, мінімум тапів.

АСПІРАЦІЙНІ міжнародні еталони категорії

  • Toggl Track — UX-еталон. Вивчити: one-tap таймер, персистентність, проста подача звітів.
  • Timely (Memory AI) — авто/AI трекінг. Вивчити: автозаповнення дня, візуал, light/dark.
  • Timeular — delightful app. Вивчити: мікровзаємодії, мотивація трекати.
  • RescueTime / Rize — automatic insights. Вивчити: дашборд-метрики, тренди.
  • Sunsama — calm productivity. Вивчити: візуальна мова, темп, відчуття простоти.
Toggl Track свідомо в аспіраційних, а не в ХАРД — для нас це UX-еталон категорії, а не вузький конкурент по фрілансерському білінгу.

04Висновки

3 спільні патерни

  1. Час монетизується через гроші — ставки + інвойси/прибутковість; голий час недостатній.
  2. Рух до зниження тертя вводу — auto-tracker і AI-перший підхід; ручний таймер вважають тертям.
  3. Freemium з per-seat апселом — безкоштовний вхід → монетизація через команди й звіти.

3 відмінності

  1. Філософія вводу: Harvest/Clockify — таймер-перші; Timely — AI-перший; TimeCamp — гібрид.
  2. Широта: Paymo — повний PM; Harvest/Timely — вузько час+білінг; Clockify/TimeCamp — +моніторинг.
  3. Модель free: TimeCamp — необмежено; Harvest/Paymo — жорсткі ліміти; Clockify урізав у 2026.

3 відкриті питання

  1. Чи не програємо тренду на авто/AI-трекінг — чи потрібен легкий авто-режим згодом?
  2. Чи достатньо MVP (PDF + статуси, 1 валюта) проти вбудованих платежів (Stripe/PayPal)?
  3. Де незайнятий простір у «mobile-first, просто для соло-фрілансера» проти Toggl/Harvest?
