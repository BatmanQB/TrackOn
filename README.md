# TrackOn

Кросплатформний веб-застосунок (адаптивний PWA) для трекінгу часу по проєктах — для фрілансерів.
Швидко затрекати час, бачити витрати часу й грошей, формувати звіти за період
та інвойси. Простий і швидкий інтерфейс; первинний контекст — десктоп, адаптив до mobile.

> **Цей README — живий індекс репозиторію.** Оновлюй його, коли з'являються нові
> матеріали чи папки.

## Можливості (MVP)

- ⏱️ **Трекінг часу** — таймер Start/Stop (персистентний, офлайн) + ручне введення
- 🗂️ **Структура** — Клієнт → Проєкт → Запис
- 💸 **Гроші** — ставка/год на проєкт (опційна), оверрайд на запис, валюти UAH/USD/EUR
- 📊 **Звіти** — на екрані, за тиждень/місяць/довільний період, по проєкту й клієнту
- 🧾 **Інвойси** — PDF + статуси оплати (не сплачено / відправлено / сплачено)
- 📱 **PWA** — встановлюється на телефон, працює офлайн
- 🌗 **Light / Dark** теми

## Стек

React + TypeScript + Vite · Tailwind CSS · PWA (vite-plugin-pwa) ·
IndexedDB через Dexie.js · Zustand

**Архітектура:** local-first зараз → синхронізація через Supabase пізніше.
